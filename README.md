## Interventions pirates

`pirate.js` — à intervalles aléatoires, **Nova 7** ou les **Renégats** détournent l'onde :
un overlay glitché s'incruste quelques secondes avec un message clandestin, puis disparaît.
(Astuce : trois appuis rapides sur `Maj` déclenchent une intrusion immédiate.)

Le module est réutilisable : `<script src="pirate.js"></script>` puis `installPirates()`.
